daisypi-bloginfo
================

Print useful analytics regarding a blog page:
- Alexa Rank
- Google Page Rank
- Zelist Rank
- Twitter followers
- Facebook page likes

Requires the rank_provider module: https://pypi.python.org/pypi/rank_provider/1.0.0

Add the input data (links to the blog, Facebook page and Twitter handler) in
the config.txt file.
